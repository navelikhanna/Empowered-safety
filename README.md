# Empowered Safety

Empowered Safety is a safety-focused web application designed to help users quickly send emergency alerts, share their location, and access nearby help during critical situations.

The project focuses on simplicity, speed, and reliability, especially in scenarios where time and accessibility are crucial.

---

## Features

- Send SOS alerts to emergency contacts
- Location sharing during emergencies
- User-friendly and responsive interface
- Backend support for alert handling and communication
- Scalable structure with separate client and server modules

---

## Tech Stack

**Frontend**
- HTML, CSS, JavaScript
- React
- Tailwind CSS

**Backend**
- Node.js
- Express.js

**Tools & Services**
- GitHub for version control
- VS Code for development
